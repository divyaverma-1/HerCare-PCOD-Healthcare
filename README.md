# 🌸 HerCare – Smart PCOD/PCOS Healthcare Platform

A full-stack healthcare management platform for women with PCOD/PCOS.

## Tech Stack

- Java 21
- Spring Boot
- Spring Security
- React
- Tailwind CSS
- MySQL
- Maven
- JWT Authentication

## Features

- User Authentication
- Period Tracker
- Symptom Tracker
- Mood Tracker
- Water Intake
- Sleep Tracker
- Exercise Tracker
- Weight & BMI
- Diet Planner
- Medication Reminder
- Doctor Consultation
- Health Reports
- Admin Panel
